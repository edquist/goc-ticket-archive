Hello

By default  MonALISA displays in this panel only the sites for which we have non zero
data about the selected VO in the selected time interval.  When the request returns, it
"unchecks"  the sites for which all the values in the selected  time interval are  zero.

In this case, if you go the Jobs per VO panel,  and by default it starts with an other VO
than LIGO, it will return data only from sites with nozero data , and "unchecks" all the
sites that do not have data ( or only zero values) for this VO
If you select now "LIGO" from the menu, you have only a subset of sites ( from the
previews request) which are selected. Please press "select all" sites before asking
for the new plot
otherwise it will show LIGO jobs only from a subset of sites that had data from the
previews request.

An other possible reason for  inconsistent  behavior may a local transparent  http proxy
that returns old cache plots (png files) ( wrongly because the directives are not to
cache them)
and returns these old plots ( from other VOs ) to the  browser.
